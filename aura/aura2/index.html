About AURA UI

Why AURA UI?

AURA UI is a curated collection of dark-themed UI components built with vanilla HTML, CSS, and JavaScript. Every component is copy-paste ready with zero dependencies — just grab the code and drop it into your project. Inspired by Aceternity UI, shadcn/ui, and Motion.dev.
